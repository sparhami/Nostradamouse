Download

There are ways to download Nostradamouse

  1. Using bower: bower install --save sparhami/Nostradamouse
  2. Directly from github: https://github.com/sparhami/Nostradamouse/tree/master/dist

Additionally, you may need the platofrm Polyfill for older browsers

  1. Using bower: bower install --save Polymer/platform
  2. Directly from github: https://github.com/Polymer/platform

Include it in Your Page

<html> <head> <!-- Load platform.js for custom element polyfill --> <script src="bower_components/platform/platform.js"></script> <!-- Load Nostradamouse --> <script src="bower_components/Nostradamouse/dist/nmouse.js" defer> </script> ... </head> ... </html>

Add Some Rules

<html> ... <body> ... <!-- nmouse rules --> <nmouse-rule src="bower_components/pretty-js/pretty-js.html"> <nmouse-trigger type="added" tagName="pretty-js"> </nmouse-trigger> </nmouse-rule> <!-- end nmouse rules --> </body> </html>
