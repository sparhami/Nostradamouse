Nostradamouse
=============

A JavaScript library to help with lazy-loading custom elements

See: http://sparhami.github.io/Nostradamouse for more information

## Getting Started

### To build

1. npm install
2. bower install
3. grunt

### To build and run the demo page

1. cd <demo name>
2. npm install
3. bower install
4. node server.js
5. go to http://localhost:8080/ using Chrome
